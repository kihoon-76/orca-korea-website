# ORCA KOREA Website

Modern corporate website for ORCA KOREA, a Korea-based bioenergy, biofuel feedstock, and global trading company.

## Tech Stack

- Next.js
- TypeScript
- Tailwind CSS
- Static content first, no backend required

## Local Development

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open `http://localhost:3000`.

## Production Build

```bash
npm run build
```

Start the production server:

```bash
npm run start
```

## Deployment

This project is ready for static deployment on Cloudflare Pages or Vercel.

Recommended build command:

```bash
npm run build
```

Recommended output directory for Cloudflare Pages:

```bash
out
```

## Notes

- The contact form is currently a static UI and should be connected to a form handler before production launch.
- Product and sustainability wording is intentionally cautious and does not claim certifications, capacities, or partnerships unless confirmed.
